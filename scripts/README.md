# Local Scripts

These PowerShell helpers are basic commands for local Docker, build, lint, and test tasks.

- Use `compose-up.ps1` and `compose-down.ps1` to start or stop the stack.
- Use `check-api.ps1`, `check-frontend.ps1`, and `verify-stack.ps1` for quick smoke checks.
- Use `ci-backend.ps1`, `ci-frontend.ps1`, and `ci-all.ps1` to repeat the simple CI flow locally.
